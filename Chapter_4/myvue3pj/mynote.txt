Scoped က component တွေ ခွဲ ပြီးတော့ CSS တွေကို သုံးတဲ့ အခါမှာ 
component ချင်း ညိနေတာ ဖြစ်နေတဲ့ အတွက် 
ကို့ css ကို ကို့ component ထဲမှာ ပဲ သုံးခွင့်ရှိတယ် လို့ ကြေငြာ ချင်တဲ့ အခါမှာ သုံးတယ်။

============================================
props က parent to child ကို data လှမ်း ပို့ချင်ရင် သုံးတယ်။
props တွေက ကြိုက်သလောက် data တွေလှမ်းပို့လို့ရတယ်
